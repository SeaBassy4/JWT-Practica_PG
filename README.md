# JWT-Practica_PG

A practice project for implementing secure authentication. It uses JSON Web Tokens (JWT) generated on the backend and stored in the browser using httpOnly Cookies to enhance security and mitigate XSS attacks.

## Tech Stack

- Node.js
- Express
- JSON Web Tokens (JWT)
- Cookie-parser
- Deployment: **Vercel**
